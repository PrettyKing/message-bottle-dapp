## 🛠️ 开发环境设置

### 本地开发
```bash
# 1. 克隆仓库
git clone https://github.com/PrettyKing/message-bottle-dapp.git
cd message-bottle-dapp

# 2. 安装依赖
npm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件

# 4. 启动开发服务器
npm run dev
```

### 合约部署
```bash
# 编译合约
npm run compile

# 部署到Monad测试网
npm run deploy

# 验证合约
npm run verify
```

### Envio索引器
```bash
# 生成代码
npm run envio:codegen

# 启动开发模式
npm run envio:dev
```
